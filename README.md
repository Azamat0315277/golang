# Golang Pet Projects

A collection of Go projects for learning and practicing various frameworks, patterns, and architectural concepts.

## Projects

### 1. **url-shortener**
URL shortening service with Redis-backed storage and REST API for creating and resolving short URLs.

### 2. **hr-system**
Human Resource Management System with MongoDB for managing employee records (CRUD operations).

### 3. **jwt-auth**
JWT-based authentication system demonstrating secure API authentication and protected routes.

### 4. **load-balancer**
HTTP load balancer implementing round-robin distribution algorithm with health checks for backend servers.

### 5. **restaurant-management**
Comprehensive restaurant management system with order tracking, menu management, table reservations, and invoicing.

### 6. **simple-crm**
Customer Relationship Management system for lead tracking and management using SQLite database.

### 7. **weather-tracker**
Weather information service integrating with OpenWeatherMap API to fetch current weather data by city.

### 8. **custom-db**
Custom file-based database engine with REST API interface for basic CRUD operations.

## Technologies Used

- **Frameworks**: Fiber, Gin, net/http
- **Databases**: MongoDB, Redis, SQLite, File-based storage
- **Auth**: JWT (JSON Web Tokens)
- **ORMs**: GORM

## Purpose

These projects are training implementations designed to explore different aspects of Go development including:
- Web frameworks and REST APIs
- Database integration and ORM usage
- Authentication and authorization
- Distributed systems concepts
- External API integration
